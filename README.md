# FiveM Mobile Shop

## Overview
FiveM Mobile Shop is a mobile application connected to a FiveM game server that allows players to purchase in-game items directly from their mobile device. Purchased items are delivered instantly to the player’s in-game account.

## Features
- Mobile shop for purchasing in-game items
- Direct integration with the game server
- Real-time delivery of items to player accounts
- Separate databases for the mobile app and the game server
- Cross-platform support (mobile and web)

## How It Works
1. Players browse and purchase items through the mobile app
2. The app stores purchase data in its own database
3. The backend synchronizes with the game server database
4. Items are automatically delivered to the player’s in-game account

## Tech Stack
- Mobile App: Flutter
- Backend: Server-side API
- Databases: Separate databases for the app and the game server
- Game Platform: FiveM

## Purpose
This project was built to demonstrate:
- Full-stack integration between a mobile app and a game server
- Database synchronization across multiple systems
- Real-time data handling and item delivery

This is a personal project created for learning and portfolio purposes.
